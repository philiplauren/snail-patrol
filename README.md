# Snail Patrol v0.2

Uppdaterad prototyp med riktig karta via Leaflet och OpenStreetMap.

## Starta

Öppna terminalen i projektmappen och kör:

```text
npm install
npm run dev
```

Öppna sedan adressen som visas, oftast:

```text
http://localhost:5173/
```

## Funktioner

- Riktig karta
- Zoom och panorering
- Snigel-pins med popup
- Registrera räddning med bild
- GPS-position
- Klicka i kartan för att välja plats
- Ranking
- Profil
- Lokal lagring i webbläsaren

Nästa steg är GitHub, Vercel och Supabase.
